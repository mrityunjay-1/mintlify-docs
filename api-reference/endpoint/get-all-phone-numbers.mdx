# Get All Phone Numbers

Retrieve a paginated list of all phone numbers in the system.

**Method:** `GET`

**Endpoint:** `/phoneNumbers`

**Query Parameters:**
- `skip` (default: 50): Number of records to skip (for pagination)
- `limit` (default: 50): Maximum number of phone number records to return

## Example Request

```http
GET {{server_url}}/phoneNumbers?skip=50&limit=50
```

## cURL Example

```bash
curl -X GET "{{server_url}}/phoneNumbers?skip=50&limit=50"
```

## Node.js Example

```js
const fetch = require('node-fetch');
fetch("{{server_url}}/phoneNumbers?skip=50&limit=50")
    .then(res => res.json())
    .then(data => console.log(data));
```

## Python Example

```python
import requests
response = requests.get("{{server_url}}/phoneNumbers", params={"skip": 50, "limit": 50})
print(response.json())
```
