# Get API Logs

Fetch detailed logs for a specific API call by callId.

**Method:** `GET`

**Endpoint:** `/apiLogs`

**Query Parameters:**
- `callId`: Unique identifier for the API call

## Example Request

```http
GET {{server_url}}/apiLogs?callId=845411f1-bc9e-4e01-865e-9ba04c8b3882
```

## cURL Example

```bash
curl -X GET "{{server_url}}/apiLogs?callId=845411f1-bc9e-4e01-865e-9ba04c8b3882"
```

## Node.js Example

```js
const fetch = require('node-fetch');
fetch("{{server_url}}/apiLogs?callId=845411f1-bc9e-4e01-865e-9ba04c8b3882")
    .then(res => res.json())
    .then(data => console.log(data));
```

## Python Example

```python
import requests
response = requests.get("{{server_url}}/apiLogs", params={"callId": "845411f1-bc9e-4e01-865e-9ba04c8b3882"})
print(response.json())
```
