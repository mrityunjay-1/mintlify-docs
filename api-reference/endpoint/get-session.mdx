# Get Session

Retrieve details for a specific session by callId.

**Method:** `GET`

**Endpoint:** `/session`

**Query Parameters:**
- `callId`: Unique identifier for the session

## Example Request

```http
GET {{server_url}}/session?callId=62575582-240f-46e7-83b9-cab19c192fd8
```

## cURL Example

```bash
curl -X GET "{{server_url}}/session?callId=62575582-240f-46e7-83b9-cab19c192fd8"
```

## Node.js Example

```js
const fetch = require('node-fetch');
fetch("{{server_url}}/session?callId=62575582-240f-46e7-83b9-cab19c192fd8")
    .then(res => res.json())
    .then(data => console.log(data));
```

## Python Example

```python
import requests
response = requests.get("{{server_url}}/session", params={"callId": "62575582-240f-46e7-83b9-cab19c192fd8"})
print(response.json())
```
